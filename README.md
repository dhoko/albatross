Simple feed reader. RSS FTW !




# Install

1. Fork the project and pull down your copy
1. Run `npm install`
1. Run `bower install`

# Run

**Dev** : `grunt dev`
> grunt dev to start watching for file saves & run a local web server at port 9000 (localhost:9000)
